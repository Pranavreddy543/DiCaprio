## Approach Framework

1. **Understand** — Restate problem, clarify constraints
2. **Plan** — Brute force first, then optimize
3. **Implement** — Clean code, handle edge cases
4. **Test** — Examples, edge cases, complexity

## Complexity Cheat Sheet

| Operation | Array | Linked List | Hash Map | BST |
|-----------|-------|-------------|----------|-----|
| Access | O(1) | O(n) | O(1) avg | O(log n) |
| Search | O(n) | O(n) | O(1) avg | O(log n) |
| Insert | O(n) | O(1) | O(1) avg | O(log n) |
| Delete | O(n) | O(1) | O(1) avg | O(log n) |
